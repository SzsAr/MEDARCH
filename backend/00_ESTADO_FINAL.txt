CAMBIO TÉCNICO PRINCIPAL
═══════════════════════════════════════════════════════════════════════════════

PROBLEMA:
  Python 3.10+ permite: list[T], dict[K,V], tuple[T]
  Python 3.8/3.9 NO lo permite → TypeError

SOLUCIÓN:
  Usar typing.List[T], typing.Dict[K,V], typing.Tuple[T]
  Funciona en Python 3.8+
